  • Atributos de la etiqueta AUDIO
  • autoplay : ejecuta el archivo al cargar la pagina
  • controls : habilita los controles del reproductos
  • loop : reproduce el archivo de manera repetitiva
  • preload : precarga el archivo antes de reproducirlo
  • src : source del archivo a reproducir
